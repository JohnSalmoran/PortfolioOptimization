⏱️ PASO 1: INSTALACIÓN (5 MINUTOS)
════════════════════════════════════════════════════════════════════════════════

OPCIÓN A: AUTOMÁTICA (Windows)
  1. Haz doble-click en:    install.bat
  2. Espera a que termine
  3. ¡Listo!

OPCIÓN B: AUTOMÁTICA (Mac/Linux)
  1. Abre terminal en esta carpeta
  2. Ejecuta:               chmod +x install.sh && ./install.sh
  3. ¡Listo!

OPCIÓN C: MANUAL
  1. Abre PowerShell/Terminal en esta carpeta
  2. Ejecuta:
     
     python -m venv venv
     
  3. Activa el entorno:
     
     Windows:     venv\Scripts\activate
     Mac/Linux:   source venv/bin/activate
     
  4. Instala dependencias:
     
     pip install -r requirements.txt
     
  5. ¡Listo!

⚠️ IMPORTANTE: LIMITACIONES
════════════════════════════════════════════════════════════════════════════════

Este es un MODELO EDUCATIVO. Algunas limitaciones:

❌ NO incluye costos de transacción (hay en realidad)
❌ NO incluye impuestos (a considerar)
❌ Asume correlaciones futuras = históricas (pueden cambiar)
❌ Asume rendimientos normales (NO siempre cierto)
❌ NO es único criterio para invertir (requiere asesoría)

✅ USA COMO:
   • Herramienta educativa
   • Análisis de apoyo
   • Complemento a asesoría profesional
   • Base para aprender Markowitz

✅ NO USES COMO:
   • Única decisión de inversión
   • Garantía de rendimiento
   • Predicción del futuro
   • Reemplazo de asesor financiero

📞 SOPORTE / AYUDA
════════════════════════════════════════════════════════════════════════════════

SI TIENES PROBLEMAS:

1. Verifica README.md → Sección "Troubleshooting"
2. Ejecuta: python test.py → Verifica dependencias
3. Intenta instalar desde cero: pip install -r requirements.txt --force-reinstall
4. Lee QUICKSTART.md → Preguntas frecuentes

ERRORES COMUNES:

❌ "ModuleNotFoundError: No module named 'streamlit'"
   ✅ Solución: pip install -r requirements.txt

❌ "No data found for ticker XXXX"
   ✅ Solución: Usa tickers válidos (ej: AAPL, MSFT, GOOGL)

❌ "Port 8501 already in use"
   ✅ Solución: streamlit run app.py --server.port 8502

❌ "yfinance error"
   ✅ Solución: pip install --upgrade yfinance
