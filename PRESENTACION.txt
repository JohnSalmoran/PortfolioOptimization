🎯 OBJETIVO DEL PROYECTO
════════════════════════════════════════════════════════════════════════════════

Crear un sistema completo de análisis cuantitativo de portafolios financieros
con interfaz gráfica profesional que permita:

  ✅ Seleccionar activos dinámicamente
  ✅ Ajustar parámetros en tiempo real
  ✅ Visualizar resultados instantáneamente
  ✅ Exportar reportes automatizados
  ✅ Comparar escenarios (what-if)

💻 TECNOLOGÍA
════════════════════════════════════════════════════════════════════════════════

LENGUAJE:
  • Python 3.12.10 (recomendado)

LIBRERÍAS CORE:
  • streamlit 1.28.0+      → GUI interactiva
  • pandas 2.0.0+          → Manipulación de datos
  • numpy 1.24.0+          → Cálculos matemáticos
  • scipy 1.11.0+          → Optimización (cuadrática)
  • yfinance 0.2.30+       → Datos Yahoo Finance
  • plotly 5.17.0+         → Gráficos interactivos
  • openpyxl 3.1.0+        → Exportación Excel

ARQUITECTURA:
  • Frontend: Streamlit + Plotly + HTML/CSS
  • Backend: Python puro + scipy.optimize
  • Datos: Yahoo Finance API
  • Almacenamiento: Local (CSV)

ALGORITMOS:
  • Optimización Cuadrática (QP)
  • Programación Convexa (scipy)
  • Simulación Montecarlo
  • Análisis de correlación
  • Pruebas estadísticas

⚠️ LIMITACIONES A CONOCER
════════════════════════════════════════════════════════════════════════════════

SUPUESTOS DEL MODELO:
  ⚠️ Rendimientos distribuidos normalmente (NO siempre cierto)
  ⚠️ Mercados eficientes (idealistico)
  ⚠️ Sin costos de transacción (en realidad los hay)
  ⚠️ Sin impuestos (a considerar)
  ⚠️ Correlaciones futuras = históricas (PUEDE CAMBIAR)

REALIDADES PRÁCTICAS:
  ⚠️ Datos históricos ≠ futuro
  ⚠️ Correlaciones suben en crisis de mercado
  ⚠️ Requiere rebalanceo periódico
  ⚠️ Comisiones reducen rendimiento
  ⚠️ NO usar como única decisión (es herramienta educativa)

RECOMENDACIONES:
  ✅ Usar como herramienta de análisis
  ✅ Complementar con asesoría profesional
  ✅ Validar supuestos
  ✅ Monitorear cambios de mercado
  ✅ Rebalancear regularmente
